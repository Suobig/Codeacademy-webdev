My Common CSS Properties Cheat Sheet

Properties for Working with Text
Tag Name Description
font-family Font Family Sets up font family for selected element(s) (default: Times New Roman)
font-size Font Size Set up font size for selected element(s)
font-weight Font Weight How bold or thin text will appear (deafult: normal)
text-align Text Align Changes text alignment of the site. Default: left. Possible values: left, right, center
